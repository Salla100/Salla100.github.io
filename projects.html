Precise Orbit Determination

Developed an algorithm at ESA ESOC using GNSS data for precise orbit determination on the OPS-SAT mission. Python-based pipeline with real satellite telemetry.

Simulated ship emergency beacons relayed through low-earth-orbit satellites using GNU Radio, Python, and ML models for signal classification.
